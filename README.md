# Doom3 MD5Model Loader


Doom3 MD5 format model

Only works with Version10

Used libs:
----------
* PyGame : Framework, Loading Image
* PyOpenGL : OpenGL 3D
* numpy : Numeric Array

Test Program:
-------------
* An example MD5Mesh with MD5Anim
* PyGame+PyOpenGL Render System
